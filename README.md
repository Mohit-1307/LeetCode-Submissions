## Overview

This repository serves as a personal log of my DSA problem-solving progress on [LeetCode](https://leetcode.com). Every accepted submission is automatically pushed here using LeetSync, organized by topic and difficulty for easy navigation.

---

## Repository Structure

Solutions are organized by topic folder and problem slug. Each submission is versioned as a separate file:

```
<topic>/
  <problem-slug>/
    submission-0.<ext>    ← initial submission
    submission-1.<ext>    ← revised submission
    ...
```

**Example:**
```
arrays/two-sum/submission-0.py
binary-search/search-in-rotated-array/submission-0.py
dynamic-programming/climbing-stairs/submission-0.py
```

---

## How Sync Works

This repository is powered by the **LeetSync** Chrome extension, which uses the LeetCode API to fetch submission data and the GitHub API to push it automatically.

```
Solve problem on LeetCode
        ↓
LeetSync detects accepted submission
        ↓
GitHub API creates or updates file in this repo
        ↓
Commit appears automatically — no manual push needed
```

**Setup (one-time):**
1. Install [LeetSync](https://chromewebstore.google.com/detail/leetsync-leetcode-to-gith/ppkbejeolfcbaomanmbpjdbkfcjfhjnd) from the Chrome Web Store
2. Click the extension icon → authorize via GitHub
3. Log in to LeetCode if prompted
4. Select this repository as the sync target
5. Start solving — submissions sync automatically
